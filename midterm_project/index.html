BITCOIN


















The site provides details of what is known as "BITCOIN" :)


Bitcoin is a cryptocurrency created by anonymous user Satoshi Nakamoto using Open-Source Software in 2008, and the currency started trading in 2009. Bitcoin is "a currency built upon computer science, economics, and cryptography". The orginal price for Bitcoin trading was $0.0008 per coin in 2010. Now the value has gone up to $56,380 (March 15th, 2021).

For the recent years, more and more new cryptocurrencies are beign developed with additional features such as smart contracts and private blockchains. Here are some examples "Ethereum (ETH), Litecoin (LTC), Bitcoin Cash (BCH), DogeCoin (DOGE)"
